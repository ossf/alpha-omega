# Erlang Ecosystem Foundation Update — April 2026

## EEF CNA: Vulnerability Report Influx

The EEF CNA has seen a significant surge in reported vulnerabilities this month,
requiring meaningful time to triage and process. This has meant some of the
ongoing work below is advancing more gradually than anticipated.

## SDLC Documentation

Security documentation and threat modeling work is in progress. A pull request
has been opened in the Hex specifications repository and is expected to be merged
in May.

* [hexpm/specifications#49](https://github.com/hexpm/specifications/pull/49)

## Rebar3 OAuth2 Support

Implementation of OAuth2 Device Grant authentication for rebar3 is underway
across multiple repositories. The work requires reviews from a wide range of
maintainers.

* [hexpm/hex_core#178](https://github.com/hexpm/hex_core/pull/178)
* [hexpm/hex#1143](https://github.com/hexpm/hex/pull/1143)
* [erlang/rebar3#3008](https://github.com/erlang/rebar3/pull/3008)
* [erlef/rebar3_hex#361](https://github.com/erlef/rebar3_hex/pull/361)

## HexDocs Per-Package Origin Isolation

Work on isolating hexdocs.pm package documentation to separate origins is in
progress. A new domain is required for private organizations, as `org.hexdocs.pm`
conflicts with `package.hexdocs.pm`. The migration has been announced to private
plan customers and is expected to be completed in May.
