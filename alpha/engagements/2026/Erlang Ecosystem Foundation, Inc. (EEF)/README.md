# Alpha Engagement: Erlang Ecosystem Foundation (EEF)

The purpose of this Alpha engagement is to fund a comprehensive security audit
of Hex.pm, the package manager and registry for the BEAM ecosystem (Erlang,
Elixir, Gleam), including:

* White-box penetration testing by Paraxial.io (Michael Lubas)
* Red-team security assessment by zentrust partners GmbH (Sven Faßbender, Martin Tschirsich)
* Remediation of identified vulnerabilities
* Threat model and SDLC documentation updates

## Timeline

This engagement starts in December 2025 (preparation phase), with audit
execution in January 2026 and remediation through March 2026.
