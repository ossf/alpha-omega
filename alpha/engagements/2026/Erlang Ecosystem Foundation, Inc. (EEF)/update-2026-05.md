# Erlang Ecosystem Foundation Update — May 2026

## EEF CNA: Vulnerability Report Influx

The surge in vulnerability reports to the EEF CNA noted last month has further
intensified, continuing to require significant time for triage and processing.
This continues to divert attention from the remaining engagement tasks and is
the reason for the extended timeline.

## SDLC Documentation

The security documentation, threat model, and SDLC docs have been completed and
published in the Hex specifications repository.

* [hexpm/specifications: Security Documentation](https://github.com/hexpm/specifications/blob/main/security/README.md)

## HexDocs Per-Package Origin Isolation

The per-package origin isolation for hexdocs.pm has been completed. Open source
package documentation is now served at `package.hexdocs.pm`, and private
organizations have been migrated to `org.hexorgs.pm` to avoid domain conflicts.

## Rebar3 OAuth2 Support

The OAuth2 Device Grant authentication implementation for rebar3 remains in PR
review across multiple repositories.
