
# OpenJS Foundation 2026 Alpha-Omega Grant

## Summary

Building on the foundation established through 2024–2025 Alpha-Omega funding, the 2026 Grant 
is focused on strengthening the security of Node.js, the critical infrastructure supporting millions of
production systems worldwide.

## 2026 Goal

Maintain and enhance Node.js as a secure, stable, and trusted foundation for global applications
through mature security processes, proactive vulnerability management, and continuous improvement
of built-in security capabilities. Establish measurable security metrics and provide active security
leadership that strengthens Node.js' role as critical infrastructure.
