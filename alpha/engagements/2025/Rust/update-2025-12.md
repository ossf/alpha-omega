## Other Engineering

### Signing / TUF Review

Walter is working on an EOY review for the work around TUF and signing. This should be available in January.

### crates.io moving to Svelte

Tobias is migrating crates.io from EmberJS to Svelte. From a security perspective, we believe this will reduce the potential for XSS vulnerabilities that may exist in Ember add-ons, as the maintenance story of Ember ecosystem is difficult right now. (Also, the typescript integration is way better in Svelte).

### GitHub environments configured as IaC

The [`team` repo](https://github.com/rust-lang/team) now defines all the GitHub environments of the repositories of the Rust Project. This improves security, as it ensures that there are drifts in the GitHub settings due to manual changes.

### docs.rs now uses Datadog for their observability.

Led by Marco, the Infra team supported the docs-rs team to move all their metrics, dashboards and alerts from the deprecated self-hosted monitoring solution to Datadog, which is more reliable, secure and offers a better developer experience.
